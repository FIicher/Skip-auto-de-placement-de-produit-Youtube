  <H3>Auto Skip Sponsor – YouTube</H3>
  <h4>Script Tampermonkey qui détecte automatiquement les passages sponsorisés dans les vidéos YouTube et les ignore instantanément grâce à l’API SponsorBlock.</h4>
</DIV>

--- 

<b>Installation :</b><br>
1- <i><a href="https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo">Installer l’extension Tampermonkey sur votre navigateur</a><br></i>
2- <i>Créer un nouveau script et y coller le contenu du fichier <code>auto-skip-sponsor.user.js</code><br></i>
3- <i>Enregistrer le script<br></i>
4- <i>Lancer une vidéo YouTube : les segments sponsorisés sont automatiquement détectés et skip instantanément<br></i>

---

<DIV align="center">

  <h4>Fini les : Cette vidéo est sponsorisée par Nord VPN et toutyquenty. Maintenant, profitez du contenu sans devenir vous-même le produit.</h4>
   <h5>Ce script utilise l’API publique SponsorBlock. Aucun compte, aucune configuration requise.</h5>

</DIV>

Conseil starter pack :
Combinez le skip automatique des sponsors YouTube avec un bon adblock, puis ajoutez l’extension <a href="https://github.com/FIicher/Supprimer-Shorts-Youtube">anti-Shorts YouTube</a>. Et voilà : vous êtes parfaitement équipé pour profiter de YouTube sans la moindre nuisance.
